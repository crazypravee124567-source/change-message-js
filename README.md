# DOM Manipulation - Beginner Project

About
A simple HTML/CSS/JavaScript project that demonstrates
basic DOM manipulation using `getElementById` and `innerHTML`.

How to Run
1. Copy the code into a file named `index.html`
2. Open the file in any browser (Chrome, Firefox, etc.)
3. Click the **"Change Message"** button to see the DOM update live

 File Structure
index.html   → Contains HTML structure + JavaScript logic

 Concepts Used
- `document.getElementById()` → Selects an element by its ID
- `.innerHTML` → Changes the content inside an element
- `onclick` → Triggers a function on button click

 What You'll Learn
- How HTML elements are selected in JavaScript
- How to change page content without reloading
- Basic event handling with buttons

Tech Stack
- HTML5
- JavaScript (Vanilla)
